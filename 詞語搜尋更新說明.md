### 📋 測試步驟

1. **打開應用程式**：http://localhost:3000

2. **切換到詞語搜尋**：點擊「詞語」標籤

3. **測試搜尋效果**：
   ```
   搜尋詞語：江南
   預期結果：
   - 顯示「江南」詞語
   - 下方顯示「粵韻: gong1 naam4」(橙色字體)
   - 不顯示分數、來源等信息
   ```

   ```
   搜尋詞語：桃花  
   預期結果：
   - 顯示「桃花」詞語
   - 下方顯示「粵韻: tou4 faa1」(橙色字體)
   ```

   ```
   搜尋詞語：春色
   預期結果：
   - 顯示「春色」詞語  
   - 下方顯示「粵韻: ceon1 sik1」(橙色字體)
   ```

4. **點擊詞語查看詳細信息**：
   - 大標題顯示詞語
   - 只顯示「粵韻: XXX」(更大字體)
   - 沒有粵韻的詞語顯示「此詞語暫無粵韻資料」

5. **測試繁簡互搜**：
   - 搜尋「学习」應能找到「學習」
   - 搜尋「電腦」應能找到「电脑」
